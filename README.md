# Mi Entreno Pro

Proyecto React + Vite listo para publicar como app web.

## Cómo probarlo en tu ordenador
1. Instala Node.js
2. Abre terminal en esta carpeta
3. Ejecuta:
   npm install
   npm run dev

## Cómo publicarlo gratis en Vercel
1. Crea cuenta en vercel.com
2. Sube esta carpeta a GitHub o arrástrala desde Vercel
3. Vercel detectará Vite automáticamente
4. Publica y obtendrás una URL real

## Cómo instalarlo en el móvil
1. Abre la URL pública en Chrome
2. Menú > Añadir a pantalla de inicio
